# Code of Conduct

Wir erwarten eine respektvolle, konstruktive Zusammenarbeit.

## Grundsätze
- respektvoller Umgangston
- sachliche Kritik, keine persönlichen Angriffe
- Fokus auf Lösung und Nachvollziehbarkeit

## Unzulässig
- beleidigende, diskriminierende oder belästigende Inhalte
- Veröffentlichung sensitiver Daten (Personendaten, Secrets)
